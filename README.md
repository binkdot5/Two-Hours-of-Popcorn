# Two Hours of Popcorn
 Movie Collection app on MEVN stack
<br />
<br />
<i> Run server first </i>
<br />
<br />
cd /back-end
<br />
fill credentials on keys.js
<br />
npm i
<br />
npm start
<br />
<br />
<i> For front-end (vue-cli required) </i>
<br />
<br />
cd/front-end
<br />
use vue-ui OR
<br />
npm i
<br />
npm run serve
